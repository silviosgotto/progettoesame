Instructions

STEP 1:  Rhythm


The three pads on the top (orange box on the right) outline the rhythm of the song: each pad stands for a different instrument.
For each pad select the Notes Number and the Metric you prefer.

From the menu (orange box on the left) select the Rhythm Pads Sounds and the BPM.
Once done, click INIT RHYTHMIC PAD 1,2,3 to initialize rhythmic pads.

For a song without melody nor harmony, click on START SELF ORDERING.
In order to set and include them in your song, please move on to the next page for instructions.

Instructions

STEP 2:  Melody


Please select in the melody pad (orange box on the right) Notes Number and Metric as desired.

Then, select from the menu (orange box on the left), Key, Scale and Melody Wave Type.

Now click INIT MELODIC PAD.
For a song without harmony click START SELF ORDERING.
To add harmony please switch to STEP 3.

Instructions

STEP 3:  Harmony


Please select in the harmony pad (orange box on the right) Notes Number.

Then, select from the menu (orange box on the left), Harmony wave type.
Now click INIT HARMONIC PAD.

Congratulations! You can now click START SELF ORDERING and allow them to shape into a random allocation that follows your preferences.

Please note that some settings* can be adjusted while the loop is playing.
These are: BPM, Rhythm Pads Sounds, Melody Wave Type, Harmony Wave Type, Envelope and Effects.

Click PLAY and enjoy!
